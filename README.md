android_device_pantech_a800s
============================

CyanogenMod10 device tree for IM-a800s (SKY Vega LTE) - Jellybean Branch

Working:
* RIL
* Graphics
* Sound
* Wifi
* Bluetooth
* Sensors
* GPS
* Camera
* Video

Known issues:
* Can not receive MMS 
* Tethering is not working
* Bluetooth SCO is not working
* HDMI is not tested
* Properiatary softwares are not working ( DMB, etc...)
